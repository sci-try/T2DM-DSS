Iraq — Textual description

Shared gate — Severe hyperglycaemia

If catabolic symptoms are present, or HbA1c ≥ 10%, or FPG > 300 mg/dL (FPG may be entered as mg/dL or mmol/L; mmol/L is converted in the tool using × 18.018), the algorithm immediately recommends start / intensify insulin. This overrides all other branches.

Step 0 — First injectable selection

How far HbA1c is above the individualised target and BMI determine the starting injectable:

  • HbA1c less than 2% above target, BMI ≤ 30: Basal insulin & titration. 2nd-generation basal insulins (degludec, glargine U-300) preferred.
  • HbA1c less than 2% above target, BMI > 30: GLP-1 RA alone (preferred) or BI + GLP-1 RA (FRC or separately).
  • HbA1c 2% or more above target, BMI ≤ 30: BI + GLP-1 RA (FRC or separately) — or premix agents if other options are not accessible locally (premix omitted if irregular meal patterns = Yes).
  • HbA1c 2% or more above target, BMI > 30: BI + GLP-1 RA (FRC or separately).

Step 1 — Escalation from basal-only

If HbA1c remains above target on basal insulin alone (applies to the HbA1c < 2% above target / BMI ≤ 30 ladder): escalate to BI + GLP-1 RA (FRC preferably, or separately).

Step 2 — Add rapid-acting insulin

If HbA1c remains above target on GLP-1 RA alone, escalate first to BI + GLP-1 RA (FRC preferably, or separately).

If HbA1c remains above target on BI + GLP-1 RA (FRC or separately): → BI (max dose) + GLP-1 RA + Rapid-acting insulin.

Titrate basal insulin to its maximum tolerated dose; add prandial rapid-acting insulin starting at the largest meal.

Step 3 — Final intensification

If HbA1c remains above target on BI (max) + GLP-1 RA + rapid-acting insulin:

  • Basal-bolus: optimise basal + rapid-acting insulin before each main meal.
  • Premixed insulin: twice-daily premixed as a simpler alternative when meal patterns are regular; if irregular meal patterns = Yes, premix is not recommended (basal-bolus only).

Standing notes (always shown for Iraq)

  • GLP-1 RA choice should be guided by patient-specific considerations: established CVD, CKD, desired weight benefit, access and cost (affordability).
  • Basal insulin: 2nd-generation agents (degludec, glargine U-300) are preferred over older generations due to lower hypoglycaemia risk.
  • Premix agents: may be used as alternatives if other options are not accessible locally (suppressed when irregular meal patterns = Yes; then prefer FRC).

Turkey — Textual description

1) Severe hyperglycaemia override

Catabolic symptoms, HbA1c ≥ 10%, or FPG > 300 mg/dL (after unit conversion) → start / intensify insulin immediately.

2) Intensification on FRC

On FRC, target unmet → add rapid-acting insulin (may be off-label).

On FRC + rapid, target unmet → basal-bolus or premixed insulin (premix omitted if irregular meal patterns = Yes).

3) Simplification

On basal-bolus or premix with recurrent hypoglycaemia → switch to FRC.

4) Intensification on basal insulin

On basal insulin, target unmet or PPG uncontrolled → switch to FRC.

5) First injectable

  • HbA1c less than 2% above target, BMI ≤ 30 → basal insulin.
  • HbA1c less than 2% above target, BMI > 30 → FRC.
  • HbA1c 2% or more above target (any BMI) → FRC.

Turkey reimbursement note: if FRC is recommended and BMI < 35 kg/m², treatment may be out-of-pocket.

Shared logic (all countries)

Severe hyperglycaemia gate

Applied before any country-specific logic. If catabolic symptoms are present, or HbA1c ≥ 10%, or FPG > 300 mg/dL (optional FPG field; mmol/L converted with × 18.018), the engine immediately returns "Start / intensify insulin (severe hyperglycaemia)".

Irregular meal patterns

Optional Yes / No (default No). When Yes, recommendations that would offer premixed insulin as an option instead favour FRC or basal-bolus only, depending on branch (Iraq and Turkey).

Default HbA1c target

If the user does not provide an individualised HbA1c target but does provide a current HbA1c value, the engine uses a default target of 7.0% and adds a comment to the output.

How the target drives routing

The engine computes HbA1c (current) − HbA1c (target) internally. The HbA1c target field in the tool accepts the individualised target value; how far the current HbA1c is above that target is then used as the primary routing variable for first injectable selection.
